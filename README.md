# GitHub Pages対応 画像検索MVP

## 特徴

- 完全サーバーレス
- GitHub Pages対応
- HTTPS対応
- スマホカメラ対応
- Screen Capture対応
- 画像アップロード対応
- クリップボード貼り付け対応

## GitHub Pages公開方法

1. GitHub repository作成
2. 全ファイルをpush
3. Settings → Pages → Deploy from branch
4. main / root を選択
